make_null_composition_space#

libcasm.composition.make_null_composition_space(
components: list[str],
allowed_occs: list[list[str]],
vacancy_names: set[str] = {'va', 'Va', 'VA'},
tol: float = 1e-05,
) → numpy.ndarray[numpy.float64[m, n]]#

Return the null space of the species fraction space as a column vector matrix

Parameters:

  • components (list[str]) – The requested component order in the composition vectors.

  • allowed_occs (list[list[str]]) – For each sublattice, a vector of components allowed to occupy the sublattice.

  • vacancy_names (set[str]) – Set of component names that should be recognized as vacancies. An exception is raised if more than one component is a vacancy.

  • tol (double = TOL) – Floating point tolerance.

Returns:

null_composition_space – The null space of the species fraction space, as a column vector matrix. Each column corresponds to an orthogonal vector in species fraction space. Each row corresponds to a component, according to components.

Return type:

numpy.ndarray[numpy.float64[n_components, n_composition_axes]]
