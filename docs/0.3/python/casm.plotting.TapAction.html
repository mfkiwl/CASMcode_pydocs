casm.plotting.TapAction¶

class casm.plotting.TapAction(data)[source]¶

Bases: object

Enable execution of a python function when a glyph is ‘tapped’ in a bokeh plot.

Example:

p = … a Bokeh plot circles = p.circ(… A set of circles) my_tap_action = TapAction(sel) def f(sel, attrname, old, new):

if len(sel.src.selected[‘1d’][‘indices’]) == 1:
# the index of the tapped configuration: index = sel.src.selected[‘1d’][‘indices’][0] … code to execute … … basically can ignore attrname, old, new …

# call f when a circle is tapped p.add_tools(my_tap_action.tool(f, [circles]))

Attributes:

data: PlottingData instance
Use to store associated Project and Selection
py_callback: {<sel_name>:<Python function>}
Dict of callback functions with signature: f(Selection, attrname, old, new). Set the callback function using the ‘tool’ method. See ‘view_on_tap’ for an example.

data: PlottingData instance
