casm.seqquest.Relax¶

class casm.seqquest.Relax(relaxdir=None, settings=None)[source]¶

Bases: object

The Relax class contains functions for setting up, executing, and parsing a seqquest calc.

The relaxation is initialized in a directory containing seqquest files, called ‘relaxdir’. It then creates the following directory structure: …/relaxdir/

run.0/ run.1/ … run.final/

‘run.i’ directories are only created when ready. ‘run.final’ is a final constant volume run

Contains:
self.relaxdir (…/relax) self.rundir (list of …/relax/run.i) self.finaldir (…/relax/run.final)

Construct a seqquest relaxation job object.

Args:

configdir: path to seqquest relaxation directory settings: dictionary-like object containing settings, or if None, it reads

the json file: …/relaxdir/relax.json
possible settings keys are:
used by seqquest.run() function:

“ncpus”: number of ncpus to run mpi on “run_cmd”: (default, see seqquest.run) shell command to execute seqquest,

or None to use default mpirun
used by not_converging():
“run_limit”: (default 10) maximum number of runs to allow before setting
status to “not_converging”

Check if configuration is relaxed.

For now, this always returns True, as SeqQuest doesn’t have the anisotropic basis set problem that VASP has.

not_converging()[source]¶

Check if configuration is not converging.

This is called when self.rundir[-1] is complete and not a constant volume job and
self.converged() == False.

Not converging criteria: >= 10 runs without completion

status()[source]¶

Determine the status of a quest relaxation series of runs. Individual runs in the series are in directories labeled “run.0”, “run.1”, etc.

Returns a tuple: (status = “incomplete” or “complete” or “not_converging”,
task = continuedir or “relax” or “constant” or None)

The first value is the status of the entire relaxation.

The second value is the current task, where ‘continuedir’ is the path to a quest job directory that is not yet completed, “relax” indicates another volume relaxation job is required, and “constant” that a constant volume run is required
