casm.seqquest.seqquest_io.Geom¶

class casm.seqquest.seqquest_io.Geom(POS=None, seqfile=None, species=None, mini=False)[source]¶

Bases: object

Container class for geometry data

Contains:
self.natoms (int): no of atoms in the unit cell self.basis (list of Sites): atoms in the crystal/molecule/etc self.type_atoms (list of str): lists the specie names as in the POS (ex. [Mn3 Mn4]) self.type_atoms_alias (list of str): lists the *atm file names for the species self.num_atoms (list of int): lists the atoms as in the POS (ex. [1 1]) *self.charge (float): net charge of the geom, if any *self.charge_loc (vec of float): location of the point charge, if any self.coord_mode (string): coord mode of POS/geom

Construct a Geom object from ‘geom’ file

Args:

POS (string): Path to a POS file seqfile (string or buffer): Path to a lcao.in, lcao.out, or lcao.geom file OR an

open file or buffer object. Info suppliments and overrides POS file
species (dict of IndividualSpecies): Species info for atoms in the
crystal/molecule/etc
