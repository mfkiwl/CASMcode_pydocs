casm.questwrapper.Relax¶

class casm.questwrapper.Relax(configdir=None, auto=True, sort=True)[source]¶

Bases: object

The Relax class contains functions for setting up, executing, and parsing a SeqQuest relaxation.

The relaxation creates the following directory structure: config/

calctype.name/
run.0/ run.1/ … run.final/

‘run.i’ directories are only created when ready. ‘run.final’ is a final constant volume run

This automatically looks for SeqQuest settings files using:
casm.project.DirectoryStructure.settings_path_crawl
