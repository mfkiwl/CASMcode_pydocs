casm.project.write_eci¶

casm.project.write_eci(proj, eci, fit_details=None, clex=None, verbose=False)[source]¶

Write eci.json

Parameters:
  • proj (casm.project.Project instance) – The CASM project
  • eci (List[(index, value)]) – index (int): linear index of basis function value (float): ECI value
  • fit_details (Dict) – Description of the fitting method used to generate the ECI, usually as output by casm.learn.to_json
  • clex (ClexDescription instance, optional, default=proj.settings.default_clex) – Specifies where to write the ECI
